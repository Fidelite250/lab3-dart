# lab3-dart

Profile Card App

A simple interactive Flutter application that displays a profile card with a dynamic like counter. The UI updates automatically based on user interaction.

Features

Profile avatar with initials

Like counter with real-time updates

Dynamic background color changes:

Default (0–7 likes)

Popular (8–14 likes)

Superstar (15+ likes)

Message updates based on the number of likes

Debug banner disabled

Built With

Flutter

Dart

material.dart

How It Works

The application uses a StatefulWidget to manage the likeCount variable.

setState() is used to:

Update the like counter

Change the background color

Update the displayed message

Learning Objectives

Understanding StatelessWidget vs StatefulWidget

Managing state using setState()

Conditional UI rendering

Basic Flutter UI design
